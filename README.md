# ✨ AI Story Generator: Adventures of

An AI-powered storytelling platform that transforms a few photos and prompts into personalized illustrated stories for children.

Designed at the intersection of generative AI, emotional personalization, and creative storytelling, AI Story Generator helps parents create magical, customized reading experiences in minutes.

---

# 🚀 Vision

Children connect deeply with stories where they see themselves reflected.

AI Story Generator reimagines children’s storytelling by enabling parents to generate personalized adventures featuring:

* Their child as the main character
* Custom themes and interests
* AI-generated narratives
* Personalized illustrations
* Interactive reading experiences

The long-term vision is to build an AI-native storytelling platform for families, education, and childhood development.

---

# 🧠 Problem

Most children’s books are:

* Generic and non-personalized
* Expensive and slow to customize
* Limited in cultural representation and inclusivity
* Difficult to tailor for learning styles or emotional needs

Parents increasingly want:

* Personalized learning experiences
* Emotionally engaging content
* Screen-time alternatives
* Creative educational tools powered by AI

---

# 💡 Solution

AI Story Generator enables parents to create unique, personalized storybooks in minutes.

Users can:

1. Upload photos of their child
2. Add interests, preferences, and story prompts
3. Generate a fully customized story
4. Create illustrated story experiences instantly

The platform combines:

* Large Language Models (LLMs)
* AI image generation
* Story personalization pipelines
* Prompt engineering
* Child-safe content generation

# ⚙️ Local Setup

```bash
# Clone repository
git clone https://github.com/shilpashams/aistorygenerator.git

# Navigate into project
cd aistorygenerator

# Install dependencies
npm install

# Start development server
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env.local` file:

```env
OPENAI_API_KEY=your_api_key_here
```
